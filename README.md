# CareerCopilot AI - Personal Job Application System

## 🚀 Features

- 🤖 AI-powered job matching using Qwen3 8B and sentence-transformers
- 🔄 Automatic job discovery from LinkedIn and Naukri
- 📄 Resume parsing and ATS optimization
- ✅ Human-in-the-loop approval workflow
- 📧 Email tracking for application responses
- 📊 Analytics dashboard with Streamlit
- 🔔 Telegram notifications
- 🗄️ SQLite database for data persistence

## 📋 Prerequisites

- Python 3.10+
- Ollama installed locally
- Google Cloud Project with Drive, Gmail, and Sheets APIs enabled
- Telegram Bot Token
- Playwright browsers
